    2.全局变量大写

    Javascript 最大的语法缺点，可能就是全局变量对于任何一个代码块，都是可读可写。这对代码的模块化和重复使用，非常不利。

    避免使用全局变量；如果不得不使用，用大写字母表示变量名，比如UPPER_CASE。
